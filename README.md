### Нейросетевое распознавание дореволюционной русской орфографии

В данной курсовой работе рассматривается задача нейросетевого распознавания до-
революционных текстов на основе сканированных изображений страниц. Предлагается ис-
пользование OCR-системы Tesseract, методов компьютерного зрения и глубинного обучения
для повышения качества распознавания, постобработки текста и сохранения структуры ори-
гинальной верстки. Предложенный подход позволит повысить точность оцифровки и может
быть применён для автоматического анализа документов.

### Как установить и запустить?
Так как Detectron2 достаточно капризная библиотека, то для стабильности, было решено разбить пайплайн, на 2 части. Дополнительно это дает лучший контроль над процессом обработки изображения.
#### Этап детекции
Для работы парсинга лейата страницы, нужно установить Detectron2 и все необходимые зависимости:

https://detectron2.readthedocs.io/en/latest/tutorials/install.html

> Для корректной работы Detectron2 советуется устанавливать все нужные зависимости в отдельное, чистое окружение.

train.py - содержит код для дообучения модели. для запуска нужно наличие размеченного датасета в формате COCO в корне проекта (папка должна называтсья export) \
Также некоторые фреймворки для разметки данных, могут использовать некорректные пути файлов, для починки был написан скрипт fix_json.py

Сами веса дообученной модели лежат в output_books/model_final.pth

Чтобы запустить инференс, нужно положить в корень папки, указать его в в константе в файле и запустить inference.py

Выходные данные будут сохранены в папку inference_output

#### Этап OCR, коррекции и сборки
Для запуска потребуется установить библиотеки:
- pytesseract
- ollama
- reportlab
- numpy
- cv2
- PIL

Чтобы запустить скрипт line_ocr.py нужно в папку с файлом добавить папку input, куда положить page.jpg - исходник изображения и boxes.json - выход предыдущего шага. Результат будет сохранен в 
папку output.
